# Changelog

## 3.1.2

- New parameters that make it usable standalone without a theme wrapper:

onTap + readOnly — tap-to-navigate pattern (decorative search bar on home screen → pushes full search page)
textStyle / hintStyle — direct style overrides
keyboardType / inputFormatters — specialized inputs (order ID search, URL search)
height / contentPadding — sizing without theme
